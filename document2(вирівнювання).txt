1) Вирівнювання схожі з флексбоксом. Вирівнювання може задавати як паррент, так і у кожного елемента може бути вирівнювання

2) align-self; Те саме, що і флексбокс. Вирівнює елемент по вертикалі. Задається самому елементу

    - stretch: по замовчуванню. Елемент розтягується на весь парент

    - start: елемент буде зверху

    - end: елемент буде знизу

    - center: елемент буде по центру

3) justify-self; Те саме, що і флексбокс. Вирівнює елемент по горизонталі. Задається самому елементу

    - stretch: по замовчуванню. Елемент розтягується на весь парент

    - start: елемент буде зліва

    - end: елемент буде справа

    - center: елемент буде по центру

- Що стосується вирівнювання від контейнера, то тут теж прямі аналогії з флексбоксом

4) align-items; Задається контейнеру. Вирівнює елемент по вертикалі - (stretch, start, end, center)

5) justify-items; Задається контейнеру. Вирівнює елемент по горизонталі - (stretch, start, end, center)
